# Git flow
- Tải project về máy
```
git clone https://github.com/Hungs20/Timtro-ReactNative.git
```
- Tạo branch mới
```
git checkout -b features/ten_branch
```
- Xem branch đang code
```
git branch
```
- Add file
```
git add .
```
- Commit
```
git commit -m 'message'
```
- Push lên git
```
git push origin features/ten_branch
```
- Cập nhật code
```
git pull origin develop
```
- Test
```
test git
```
